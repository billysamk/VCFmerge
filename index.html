Merge all your VCF contacts into one file
by billysamk

Most of us has faced error where we used copy *.vcf to all.vcf to merge all our contacts. Although the process copies all contacts but we are faced with error when we try to upload the file to google contacts. well here is the solution for that problem

This will correct the problem where only one contact is uploaded or no contacts are uploaded to the google account
Instructions
  • 1. Download the file and place it in the folder where all your contacts are saved.
  • 2. Double click or press enter on the file
  • 3. Find the merged vcf file in C:\Temp folder
  • 4. Upload to Google contacts.
